# GKB24 MAX Bot

Сервис чат-бота для MAX, который принимает обращения пользователей, собирает контактные данные и передает заявку менеджерам и в API сайта ГКБ №24.

## Возможности

- запуск диалога через webhook MAX
- пошаговый сценарий общения с пользователем
- сбор имени, описания проблемы и контактного телефона
- защита от повторной обработки одинаковых сообщений
- отправка заявки в чат менеджеров
- отправка обращения в API сайта ГКБ №24
- healthcheck endpoint для проверки состояния сервиса

## Стек

- Node.js
- Express
- Axios
- dotenv
- PM2
